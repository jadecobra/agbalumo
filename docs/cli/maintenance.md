# agbalumo CLI: System Maintenance

The following commands are used for running, seeding, and benchmarking the system.

## Commands

### serve

Start the agbalumo web server.

```bash
agbalumo serve
```

### seed

Seed the database with sample data.

```bash
agbalumo seed
```

### benchmark

Run basic system benchmarks.

```bash
agbalumo benchmark
```

### stress

Run system stress tests.

```bash
agbalumo stress
```
